# Task 4 – Common Network Security Threats

## Oasis Infobyte Cyber Security Internship

### Project Overview

This repository contains my research report on **Common Network Security Threats**, completed as part of the **Oasis Infobyte Cyber Security Internship**.

The objective of this task is to understand how different types of network attacks work, the risks they pose to individuals and organizations, and the security measures that can be used to prevent them.

The report covers several commonly encountered network threats along with real-world examples, their impact on cybersecurity, and practical mitigation strategies.

---

## Objectives

- Understand common network security threats.
- Study how attackers exploit network vulnerabilities.
- Learn preventive measures used in modern organizations.
- Improve cybersecurity awareness and defensive thinking.

---

## Topics Covered

- Denial of Service (DoS)
- Distributed Denial of Service (DDoS)
- Man-in-the-Middle (MITM)
- IP Spoofing
- DNS Poisoning / DNS Spoofing
- Comparison of different network attacks
- Best security practices
- References from trusted cybersecurity sources
